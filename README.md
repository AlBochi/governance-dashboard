# Governance Dashboard

A board-ready AI governance dashboard template for financial institutions. Tracks model inventory, risk assessments, audit readiness, and compliance status in a single view.

## What It Does

- Visualizes complete AI model inventory with risk classifications
- Tracks regulatory compliance status across OSFI E-23 and SR 11-7
- Displays real-time audit readiness metrics
- Generates board-level summary reports

## Quick Start

git clone https://github.com/AlBochi/governance-dashboard.git
cd governance-dashboard
pip install -r requirements.txt
python dashboard.py
